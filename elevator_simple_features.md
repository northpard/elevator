# 엘리베이터 — 단순 기능표

| 기능 | 설명 |
|---|---|
| 보드 | Arduino Uno (가정) |
| 호출 버튼 | 외부 호출 버튼 3개 (1층, 2층, 3층) |
| 버튼 LED | 각 호출에 대응하는 표시 LED (버튼 옆) |
| 층 LED | 현재층을 표시하는 LED 3개 |
| 층간 LED | 이동 애니메이션용 노란색 LED 4개 |
| 호출 동작 | 짧게 누름 → 호출 등록 (버튼 LED ON) |
| 취소 동작 | 길게 누름(=취소) → 호출 취소 (버튼 LED 깜빡임) |
| 취소 보호 | 취소 직후 재등록을 짧게 차단 (글리치 방지) |
| 스케줄링 | 같은 방향 우선 처리 (SCAN 유사) |
| 설계 방식 | 논블로킹(밀리초 시간 기준) — 입력, 애니메이션, 이동 비동기 처리 |
| 미구현/제한 | 캐빈 버튼 및 EEPROM 미구현; 부팅 시 핀 글리치 가능성 있음 |

> 참고: 더 간단한 표(또는 영어 버전)를 원하시면 알려주세요.
